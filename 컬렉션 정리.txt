* Java Collection Framework

* 컬렉션의 특징
 - 크기 제한 X
 - 추가,수정,삭제,정렬 등의 기능 제공
 - 모든 객체 저장 가능
   단, 제네릭스를 이용해서 타입 제한
   
   
* 컬렉션 제공 인터페이스
- List : 인덱스를 이용해서 순서 유지, 중복 허용
- Set  : 순서 유지 X, 중복 허용 X
- Map  : K, V 쌍으로 데이터를 관리
   
* List, Set은 공통된 이름의 메서드가 많아서
 이를 추상화 하여 Collection 인터페이스를 만들어 상속
